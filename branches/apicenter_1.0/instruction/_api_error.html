API Errors Collection

API error message will be displayed if the API requested fail, for example, 1) if the user provide an invalid api_k parameter value 2) if the user request Write, Modify and Delete APIs by using GET HTTP method etc. Below is a sample error message of a failed API Request:

<?xml version="1.0" encoding="UTF-8"?><error_code>121</error_code>
<error_message>API Access Error! API Key must be 16 alphanumeric chars.</error_message>
 
{
    "error_code": "121",
    "error_message": "API Access Error! API Key must be 16 alphanumeric chars."
}
Error Output XML Data IntroductionError Output JSON Data Introduction
XML TagsJSON Nodes Parent Description
error_code root

The error_code tag represents the error identifier of the API requested, each error code represents a spcific API Reqeust error.
error_message root

The error_message tag represents the error message of the API requested, the user can figure out why the API request was failed.
